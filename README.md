# Personal Portfolio Website

My personal software engineering portfolio, showcasing my projects, skills, and experience as a Computer Science student at Oakland University.

**Live Site:** [https://vmgriest.github.io/Portfolio/](https://vmgriest.github.io/Portfolio/)

## About

This website serves as my professional online presence, highlighting:
- Software engineering projects (machine learning, web apps, Java applications)
- Technical skills (Python, Java, PHP, JavaScript, C#, C, HTML/CSS)
- Work experience (UWM Intern, SECS Teaching Assistant)
- Education and extracurriculars

## Features

- Clean, responsive design
- Project gallery with descriptions and links
- Separate pages for coursework and hobbies
- Direct links to GitHub repositories
- Contact information and social links

## Technologies Used

- HTML5
- CSS3
- GitHub Pages (hosting)

## File Structure

| File | Purpose |
|------|---------|
| `index.html` | Main landing page with projects and experience |
| `courseProject.html` | Showcase of academic coursework projects |
| `hobbies.html` | Personal interests and extracurricular activities |
| `style.css` | Main stylesheet for consistent design |
| `images/` | Directory for profile pictures, project screenshots, and icons |
| `README.md` | This documentation |
